# 2024.02.26 stock report

## 마녀공장
- 2024년 외형 회복이 본격화될 전망이다. 회사는 2012년 설립되어 자연주의 기능성 스킨케어 사업을 확대해왔고, 2018년에는 엘엔피코스메틱을 인수하며 코스닥에 상장했다. 주력 브랜드는 마녀공장이며, 해외 비중은 55%로 주로 일본에서 매출을 올리고 있다. 그러나 2023년 3분기에는 매출과 영업이익이 감소하며, 오프라인 확장 시도로 인한 비용 부담도 있었다. 동사는 일본 시장에 집중하기 위해 노력 중이며, 오프라인 확장을 위해 투자도 진행 중이다.
## 엔씨소프트
한화투자증권은 엔씨소프트의 이익이 줄어들고 모멘텀이 둔화되었다고 평가했다. 새로운 게임 라인업은 5개로 예정되어 있으며, '배틀크러쉬'와 'BSS'는 상반기에, 'TL글로벌'과 '기존IP 스핀오프'는 하반기에 출시될 예정이다. '프로젝트G'와 '아이온2'는 내년으로 연기되었다. 신작을 통한 매출 성장은 상반기까지 어려울 것으로 전망된다.
## 화승엔터프라이즈
