# 2024.03.20 stock report
## LG생활건강

## JYP Ent.
- 2024년 1분기 실적 예상을 상회했으며, 일본 매출이 이연되지 않았다면 더 좋은 성과를 보였을 것이다. 하지만 하향앨범 감소 이슈로 목표주가를 120,000원으로 하향 조정하였다. 향후 상반기에는 견조한 실적이 예상되며, 팬덤을 확보하고 음원/기타 매출이 성장하고 있다. 현재 주가는 노 재팬 이슈를 반영하여 저평가되어 있으며, 최선호주를 유지한다.
## 디와이파워

## JYP Ent.
