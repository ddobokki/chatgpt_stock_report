# 2024.06.17 stock report

## 빙그레
- 2분기 기대 호실적, 매수 추천
- 2분기 매출액 및 영업이익 예상치 상회
- 국내외 냉장과 냉동 부문의 성장세 예상

## 코스메카코리아

리포트:
