- 한화에어로스페이스는 본격적인 방산 성장의 해를 맞이하고 있으며, 4분기 2023년에는 매출액과 영업이익이 예상치를 상회하며 어닝 서프라이즈를 기록했다. 지상방산 부문의 실적 호조가 전체 매출을 견인했다. 투자의견은 매수이며, 목표주가는 210,000원이다.
